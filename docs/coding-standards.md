# Coding Standards

## Backend (Laravel)
- Follow PSR-12
- Use Service layer for business logic
- Keep controllers thin
- Use meaningful naming

## Frontend (React)
- Functional components
- Separate logic (services) from UI
- Keep components small

## General
- Small commits
- Clear naming
- No hardcoded values (use .env)
